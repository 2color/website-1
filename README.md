## Install

```sh
git clone https://github.com/multiformats/website
```

## Usage

#### Build

```sh
> hugo
```

#### Serve

```sh
> hugo serve
```

#### Deploy

```sh
> hugo
> ipfs add -r public
> open https://ipfs.io/ipfs/<last hash from above>
```
